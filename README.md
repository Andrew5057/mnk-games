# *m,n,k*-Games

An [*m,n,k*-game](https://en.wikipedia.org/wiki/M,n,k-game) is a game in which
two players compete to get *k* stones in a row on an *m*-by-*n* board. This
project aims to provide tools for analyzing (and playing) *m,n,k*-games,
including restricted *m,n,k*-games like
[Connect Four](https://en.wikipedia.org/wiki/Connect_Four).
